## 1. 评审说明

### 1.1 评审状态定义
| 状态 | 含义 |
|---|---|
| `待评审` | 尚未进入评审 |
| `通过` | 已通过，无需改动 |
| `有条件通过` | 通过但需补充改动 |
| `不通过` | 需重做后复审 |

### 1.2 评审意见栏使用规则
1. `评审意见` 写明确动作，不写“看情况”“再讨论”。
2. `负责人` 必须到人。
3. `截止日期` 必须是具体日期。
4. 所有 `有条件通过/不通过` 项必须回填 `复审结果`。
